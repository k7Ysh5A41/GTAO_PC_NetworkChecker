本项目与 Take Two 和 Rockstar Games 无任何关联。
请自行权衡风险，因为您可以通过本软件查看其他玩家的 ip 地址。

English:
This is a small project to help GTA Online for PC players to check the session's network.Users can check the Peer to Peer network and find if someone is lagging the session.
This project has no affiliation with Take Two & Rockstar Games.
Please use as your risk because you can check the ip of other players.
This project is for Simplified Chinese players.If you want to use this project in English or other languages,feel free to create a new branch :)
